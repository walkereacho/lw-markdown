# Unordered Lists

A simple list with dashes:

- First item
- Second item
- Third item

A list with asterisks:

* Apple
* Banana
* Cherry

A list with plus signs:

+ Red
+ Green
+ Blue

Nested unordered list:

- Top level item
  - Nested item one
  - Nested item two
    - Deeply nested item
    - Another deep item
  - Back to second level
- Another top level item
